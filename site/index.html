LLM Paper Review Board

Three frontier LLMs claude codex gemini independently review each paper across two cycles, then a board agent synthesizes the cross-review.

Two datasets: gr-qc — arXiv April 2026, gr-qc papers ≤10 pages; bs-meter — papers Sabine Hossenfelder rated on her bullshit-meter, scored 0–10 (10 = most bullshit).

Dataset
How this works
  • Inputs. Each paper is rendered into a single markdown bundle. Reviewers see only the bundle — no metadata, no Sabine score, no sibling papers from the same video.
  • Three independent reviewers. claude codex gemini each emit a structured JSON: claims with support level, technical objections with severity + confidence, missing assumptions, novelty assessment, points needing human spot-check, and a recommended verdict from a six-value enum.
  • Two cycles. Cycle 1 is independent. In cycle 2 each reviewer is shown the other two’s cycle-1 reports and writes a follow-up — keeping real objections, retracting weak ones, marking which of the others’ points they accept or reject, and re-scoring.
  • Final score. Each reviewer hands in one overall_score_10 per cycle. The displayed score is the mechanical mean of the three reviewers’ cycle-2 overall scores — no human weighting. The five dimension scores (technical soundness, novelty, significance, clarity, reviewer confidence) are reported per reviewer for information only and are not aggregated into the displayed score.
  • Board synthesis. A separate claude run reads all six reviewer JSONs and produces the public summary, strongest supported claims, strongest objections (with which reviewers backed each), reviewer disagreements, and suggested human spot-checks shown on each paper’s detail page.
  • bs-meter view. Sabine’s 0–10 score runs opposite the board’s: high BS-meter ↔ low quality. The agreement metric is |board + sabine_BS − 10| (small = agree, large = disagree). A few rows are papers Sabine discussed but didn’t pin down to one number — the BS column shows companion (sibling paper from the same video, score is on the primary), qualitative (she only gave a verbal verdict), or the candidate scores joined with or (e.g., 0 or 10) when she explicitly offered more than one.
